 ## Trip Haven - Travel Website [https://trip-haven.netlify.app/]
A modern, full-featured travel website built with React, designed to help users explore and book amazing destinations across Bangladesh.

 ## Features

# Home Page

Stunning hero section with call-to-action
Featured destinations showcase
Popular packages
Customer testimonials
Newsletter subscription

# Destinations Page

100+ Travel Destinations across Bangladesh
Smart Search - Real-time destination search
Category Filtering - Beach, Mountain, Nature categories
Interactive Cards - Beautiful destination cards with ratings, reviews, and pricing
Responsive Grid Layout - Optimized for all devices

# Gallery Page

Beautiful Photo Grid - Masonry-style image gallery
Lightbox Modal - Full-screen image viewing with navigation
Category Filter - Filter by destination types
Search Functionality - Find images by title or location
Smooth Animations - Hover effects and transitions

# Contact Page

Contact Form - Fully functional with validation
Google Maps Integration - Office location display
FAQ Section - Common questions and answers
Social Media Links - Facebook, Instagram, Twitter, WhatsApp
24/7 Support - Helpline and emergency contact

# About Us Page

Company Story - Journey and mission
Statistics - 50K+ travelers, 100+ destinations
Core Values - Passion, Trust, Innovation, Service
Timeline - Company milestones from 2005-2025
Team Members - Meet the professionals

# Login/Signup Page

Dual Functionality - Switch between login and signup
Social Login - Google and Facebook integration
Password Visibility Toggle - Show/hide password
Form Validation - Real-time field validation
Remember Me - Persistent login option
Responsive Design - Beautiful on all devices

# 404 Error Page

Creative Design - Animated error display
Quick Links - Navigate to important pages
Search Assistance - Helpful navigation options
Support Contact - Easy access to help

# Design System

# Color Palette

Primary Color: #00A4BB (Cyan)
Secondary Colors: Various shades of gray for text and backgrounds
Accent Colors: Gold for ratings, social media brand colors

# Typography

Font: System font stack (optimized for Bengali and English)
Headings: Bold, large sizes for impact
Body: Clear, readable text with proper line-height

# Components

Consistent button styles
Card-based layouts
Icon integration with React Icons
Smooth animations and transitions

# Technologies Used

React 18.x - Modern React with hooks
Tailwind CSS - Utility-first CSS framework
React Icons - Icon library (Font Awesome, Material Design)
React Router - Client-side routing (recommended)
JavaScript ES6+ - Modern JavaScript features
